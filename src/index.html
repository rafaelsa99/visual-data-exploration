Exploração Visual de Dados

Esta aplicação tem como objetivo a exploração visual de dados. As visualizações ilustram características relevantes dos dados e permitem ao utilizador explorar visualmente essa informação. As visualizações foram desenvolvidas utilizando a ferramenta d3.js.

Visualização de dados relativos ao MIEET e comparação com cursos semelhantes

Descrição dos Dados

O dataset contém informação relativa às candidaturas ao Concurso Nacional de Acesso ao Ensino Superior para o MIEET e para cursos congéneres. Para os vários cursos é apresentada uma evolução histórica, com dados entre 2007 e 2016.

Utilizadores e Contexto

A aplicação destina-se a pessoal docente ligado ao DETI, em particular ao MIEET. Os utilizadores poderão utilizar a aplicação durante a análise anual às candidaturas ao MIEET, ou sempre que quiserem consultar informação estatística ou comparar o MIEET com cursos congéneres.

Principais Perguntas

A aplicação deve ajudar a responder a perguntas como: Qual a distribuição de notas dos colocados no MIEET? Qual a posição do MIEET nas preferências dos candidatos? Quais as alternativas dos alunos candidatos no MIEET? Como se destaca o MIEET em comparação com cursos semelhantes?
