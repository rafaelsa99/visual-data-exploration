Notas dos Colocados

A visualização apresenta a evolução temporal das notas de candidatura dos colocados num determinado curso, bem como a distribuição dos colocados pelas várias notas.
A média dos vários anos é calculada para cada nota, sendo representada no gráfico pela linha.

As interações disponíveis na visualização são:

Seleção do intervalo temporal: Através do slider é possível selecionar um intervalo de tempo específico, dentro dos anos disponíveis, para o qual se quer visualizar os dados.

Seleção do curso: Através do menu drop-down é possível escolher o curso para o qual se quer visualizar os dados.

Filtragem de dados: Ao premir a legenda, apresentada ao lado do gráfico, é possível mostrar/ocultar informação relativa a um ano ou à média. A informação visível no gráfico é representada com a sua legenda a negrito.

Informação detalhada: Ao mover o cursor para uma barra do gráfico, é possível visualizar informação detalhada acerca do ano e do número de colocados naquele ano com a respetiva nota.
